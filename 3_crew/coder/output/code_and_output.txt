```python
# File: pi_series_calculation.py

# Function to calculate the approximation of pi using the given series
def calculate_pi_series(num_terms):
    total = 0.0
    for n in range(num_terms):
        term = (-1) ** n / (2 * n + 1)
        total += term
    return total * 4  # Multiply by 4 to get the approximation of pi

# Main function to execute the calculation
if __name__ == "__main__":
    terms = 10000
    pi_approximation = calculate_pi_series(terms)
    
    # Output the result
    print(f"The approximation of pi using the first {terms} terms is: {pi_approximation}")
```

### Output
When you run the above code, the output will be:

